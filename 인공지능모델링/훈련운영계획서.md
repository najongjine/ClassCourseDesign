## 2. 훈련목표
1.  빅데이터 분석의 핵심이 되는 머신러닝(Machine Learning)과 딥러닝(Deep Learning) 기술을 활용하여 현업의 데이터를 효율적으로 처리하고 분석할 수 있는 능력을 함양한다.
2.  Python 기반의 데이터 전처리 및 탐색적 데이터 분석(EDA) 역량을 확보하여 인공지능 모델링의 기초가 되는 양질의 데이터를 확보할 수 있다.
3.  다양한 정형/비정형 데이터를 활용하여 인공지능 모델을 설계 및 학습시키고, 최적의 하이퍼파라미터 튜닝을 통해 예측 성능을 극대화하는 실무 역량을 배양한다.
4.  인공지능 모델의 성능을 정량적으로 평가하고 지속적인 모니터링 및 유지보수를 수행할 수 있는 AI 파이프라인 운영 능력을 갖춘다.

> **※ 훈련목표 설정 배경**
> 최근 빅데이터 분석 트렌드는 단순 통계를 넘어 AI 모델링을 통한 예측 분석으로 고도화되고 있습니다. 이에 본 과정은 이론 중심의 학습을 탈피하고, **실제 빅데이터 분석 과제(Type 1, 2 유형 등)**와 유사한 시나리오를 반복 실습함으로써 훈련생이 수료 즉시 현장의 데이터 직무를 수행할 수 있도록 설계하였습니다.

## 3. 훈련대상 및 선수요건
*   **훈련대상**: 빅데이터 분석 및 AI 개발 분야로 진출하고자 하는 구직자 (비전공자 포함)
*   **선수요건**: 기본적인 컴퓨터 활용 능력 보유자 (프로그래밍 비경험자 학습 가능하도록 기초부터 구성)

## 4. 훈련운영 전략 (교수학습 방법)

### 가. 실무 밀착형 프로젝트 학습 (Project Based Learning)
*   **빅데이터 분석 실습 강화**:
    *   공공데이터포털, Kaggle 등의 실제 데이터를 활용하여 데이터 수집부터 전처리, 모델링, 평가까지의 **End-to-End 프로세스**를 체화합니다.
    *   특히 **빅데이터 분석에서 자주 다루는 정형 데이터 전처리(결측치, 이상치, 인코딩, 스케일링)** 과정을 집중 훈련하여 데이터 핸들링 자신감을 고취합니다.

### 나. 단계별 퀘스트 해결 방식 (Gamification 요소를 일부 도입)
*   훈련생이 지루하지 않게 학습할 수 있도록, 난이도별 데이터 분석 미션을 제공합니다.
    *   *Lv 1. 데이터 구멍을 메꿔라! (결측치 처리)*
    *   *Lv 2. 범인은 누구? (이상치 탐지)*
    *   *Lv 3. 최고의 모델을 찾아라 (모델 성능 경쟁)*

### 다. 지속적인 형성평가 및 피드백
*   매 모듈 종료 시 **미니 테스트(작업형 실기 형태)**를 실시하여 훈련생 본인의 코딩 실력을 객관적으로 점검합니다.
*   강사의 1:1 코드 리뷰를 통해 비효율적인 코드를 개선하고 'Pythonic'한 코드 작성을 유도합니다.

## 5. 학생 관리 및 취업 지원 방안
*   **학습 부진자 관리**: 일일 코딩 테스트 결과 하위 그룹 대상 보충 학습(Restudy) 및 멘토링 진행.
*   **포트폴리오 지도**: 수업 중 진행한 미니 프로젝트들을 엮어 개인별 직무 역량이 드러나는 GitHub 포트폴리오 완성.
*   **모의 면접**: 빅데이터/AI 직군 기술 면접 대비, 주요 알고리즘 및 CS 지식 모의 면접 진행.

## 6. 기대효과
*   데이터 문해력(Data Literacy)을 갖춘 실무형 AI 인재 양성.
*   빅데이터 분석 관련 주요 라이브러리(Pandas, Scikit-learn, Tensorflow/PyTorch) 능숙 활용 가능.
*   기업 요구사항에 부합하는 데이터 전처리 및 모델링 수행 능력 확보.
