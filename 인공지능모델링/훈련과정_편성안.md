# 훈련과정 편성안 (교과목 구성)

### 1. 교과목 구성 개요
본 과정은 **‘빅데이터 분석’** 역량을 중심으로 NCS(인공지능 모델링) 능력단위를 재구성하였습니다.
초반부에는 Python 기초 및 데이터 핸들링(비NCS)을 배치하여 기초를 다지고, 중후반부에는 NCS 능력단위를 활용하여 빅데이터 분석 기법(머신러닝, 딥러닝)을 심도 있게 다룹니다.
